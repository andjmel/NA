# 📏 Нумеричка Интеграција — Cheatsheet

> Сви кодови за нумеричку интеграцију (правоугаоник, трапез, Симпсон) у Матлабу.

---

## 0. Припрема — заједничка за све методе

```matlab
f = @(x) exp(x.^2);   % подинтегрална функција (.^ ради по елементима вектора)
a = 0;                 % доња граница
b = 1;                 % горња граница
n = 10;                % број подинтервала (за Симпсон мора бити ПАРАН)

X = linspace(a, b, n+1);   % n+1 равномерно распоређених чворова
h = (b-a)/n;               % корак интеграције

% Формирање табеле F (n+1) x 4:
%   колона 1: x вредности
%   колона 2: само крајеви f(x0) и f(xn)       → коефицијент 1
%   колона 3: непарни унутрашњи чворови          → коефицијент 4 (Симпсон)
%   колона 4: парни унутрашњи чворови            → коефицијент 2 (Симпсон)

F = zeros(n+1, 4);
F(:,1) = X;                        % сви чворови у прву колону
F(1,2) = f(X(1));                  % f(x0) — леви крај
F(n+1,2) = f(X(n+1));             % f(xn) — десни крај
F(2:2:n, 3) = f(X(2:2:n));       % непарни чворови x1,x3,... (MATLAB индекс 2,4,...)
F(3:2:n, 4) = f(X(3:2:n));       % парни унутрашњи x2,x4,... (MATLAB индекс 3,5,...)
disp(F);
```

> **Зашто овако организована табела?** Да исти низ вредности послужи за све три формуле — само се мењају коефицијенти.

## 1. ФОРМУЛА ПРАВОУГАОНИКА (левих)

**Идеја:** На сваком подинтервалу функцију апроксимирамо **константом** — вредношћу у левом крају. Интеграл константе × ширина = $f(x_{i-1}) \cdot h$.

$$\int_a^b f(x)\,dx \approx h\bigl[f(x_0) + f(x_1) + \cdots + f(x_{n-1})\bigr]$$

> $f(x_n)$ **не улази** — леви правоугаоници не користе десни крај интервала.

```matlab
Ip = h * (F(1,2) + sum(F(:,3)) + sum(F(:,4)))
% F(1,2)       = f(x0) — леви крај целог интервала
% sum(F(:,3))  = f(x1) + f(x3) + ... унутрашњи непарни
% sum(F(:,4))  = f(x2) + f(x4) + ... унутрашњи парни
% заједно: f(x0) + f(x1) + ... + f(x_{n-1}), тј. n сабирака

Ip = h * (sum(F(:,3)) + sum(F(:,4)) + F(n+1,2));
%          parni         neparni       xn (desna krajnja)
%          (bez x0)      (bez x0)
```

| Особина | Вредност |
|---|---|
| Степен тачности | 1 |
| Ред грешке | $O(h)$ — преполовиш $n$, преполовиш грешку |
| Оцена грешке | $E_p \leq \dfrac{h}{2}(b-a)M_1$ |

---

## 2. ФОРМУЛА ТРАПЕЗА

**Идеја:** На сваком подинтервалу апроксимирамо функцију **правом линијом** кроз оба краја. Трапез са основицама $f(x_{i-1})$ и $f(x_i)$ и висином $h$ има површину $\frac{f(x_{i-1})+f(x_i)}{2} \cdot h$. При сабирању, унутрашњи чворови улазе двапут (из два суседна трапеза), па добијају коефицијент 1, а крајеви само по $\frac{1}{2}$.

$$\int_a^b f(x)\,dx \approx h\left[\frac{f(x_0)+f(x_n)}{2} + f(x_1) + f(x_2) + \cdots + f(x_{n-1})\right]$$

```matlab
It = h * ((F(1,2) + F(n+1,2))/2 + sum(F(:,3)) + sum(F(:,4)))
% (F(1,2) + F(n+1,2))/2  = (f(x0) + f(xn)) / 2 — крајеви са половином тежине
% sum(F(:,3)) + sum(F(:,4)) = сви унутрашњи чворови са пуном тежином
% разлика vs правоугаоник: f(xn) УЛАЗИ, али крајеви деле са 2
```

| Особина | Вредност |
|---|---|
| Степен тачности | 1 |
| Ред грешке | $O(h^2)$ — удвостручи $n$, грешка се **4× смањи** |
| Оцена грешке | $E_t \leq \dfrac{(b-a)}{12}h^2 M_2$ |

---

## 3. СИМПСОНОВА ФОРМУЛА ⭐

**Идеја:** На сваком **пару** подинтервала апроксимирамо функцију **параболом** кроз три тачке. Интеграл параболе кроз три равномерне тачке аналитички даје шему 1-4-1. При сабирању парабола, непарни чворови (средине парабола) добијају коефицијент **4**, парни унутрашњи (деле две суседне параболе) добијају **2**, крајеви добијају **1**.

$$\int_a^b f(x)\,dx \approx \frac{h}{3}\left[f(x_0) + f(x_n) + 4\sum_{\text{непарни}} f(x_i) + 2\sum_{\text{парни унутр.}} f(x_i)\right]$$

> ⚠️ **$n$ мора бити паран** — свака парабола покрива два подинтервала.

```matlab
Is = h/3 * (F(1,2) + F(n+1,2) + 4*sum(F(:,3)) + 2*sum(F(:,4)))
% F(1,2) + F(n+1,2)  = f(x0) + f(xn)         → коефицијент 1
% 4*sum(F(:,3))       = 4*(f(x1)+f(x3)+...)   → коефицијент 4 (средине парабола)
% 2*sum(F(:,4))       = 2*(f(x2)+f(x4)+...)   → коефицијент 2 (заједничке тачке)
```

| Особина | Вредност |
|---|---|
| Степен тачности | 3 (иако је парабола, симетрија поништава члан са $f'''$) |
| Ред грешке | $O(h^4)$ — удвостручи $n$, грешка се **16× смањи** |
| Оцена грешке | $E_s \leq \dfrac{(b-a)}{180}h^4 M_4$ |

## 5. ТЕОРИЈСКА ОЦЕНА ГРЕШКЕ

```matlab
syms x
f_sym = exp(x^2);          % симболичка функција (упиши своју)

for i = 1:5
    df = diff(f_sym, i);   % рачуна и исписује i-ти извод
    disp(df)               % из исписа очитаваш вредности за M1, M2, M4
end

% Максимуме Mi одреди тако што уврстиш krajeve intervala u i-ti izvod
% i узмеш већу вредност (за rastuće funkcije to je obično desni kraj b)

M1 = 2*exp(1);    % max|f'(x)|   na [a,b] — очитано из симболичког извода
M2 = 6*exp(1);    % max|f''(x)|  na [a,b]
M4 = 76*exp(1);   % max|f''''(x)| na [a,b]

Ep = h/2 * (b-a) * M1          % оцена грешке правоугаоника
Et = (b-a)/12 * h^2 * M2       % оцена грешке трапеза
Es = (b-a)/180 * h^4 * M4      % оцена грешке Симпсона
```

> **Зашто M4 а не M3 за Симпсон?** Јер је формула симетрична — члан са трећим изводом се поништава, па грешка зависи од четвртог извода.

## 6. РУНГЕОВА ОЦЕНА ГРЕШКЕ ⭐

**Идеја:** Кад не знамо изводе, рачунамо интеграл двапут — фином мрежом (корак $h$) и грубом (корак $2h$) — и из разлике процењујемо грешку:

$$E \approx \frac{|I_h - I_{2h}|}{2^p - 1}$$

где је $p$ ред методе. За Симпсон $p=4$, па је делилац $2^4-1=15$.

```matlab
p = 4;    % ред методе: правоугаоник=1, трапез=2, Симпсон=4

% Фина мрежа (корак h, n подинтервала) — исти Is као горе:
Ihs = h/3 * (F(1,2) + F(n+1,2) + 4*sum(F(:,3)) + 2*sum(F(:,4)))

% Груба мрежа (корак 2h, n/2 подинтервала) — користимо само сваки други чвор:
% нови чворови су: x0, x2, x4, x6, x8 (од нула) тј. X(1),X(3),X(5),X(7),X(9)
% у Симпсону на грубој мрежи:
%   крајеви:          x0 и xn           → sum(F(:,2))
%   коефицијент 4:    x2, x6, ...       → F(3:4:n, 4), тј. MATLAB индекси 3,7,...
%   коефицијент 2:    x4, x8, ...       → F(5:4:n, 4), тј. MATLAB индекси 5,9,...

I2hs = (2*h)/3 * (sum(F(:,2)) + 4*sum(F(3:4:n,4)) + 2*sum(F(5:4:n,4)))
%formula pravougaonika
I2hp = (2*h)*(F(1,2)+sum(F(:,4)));
 
%formula trapeza
I2ht = 2*h*((F(1,2)+F(n+1,2))/2+sum(F(:,4)));

E = abs(Ihs - I2hs) / (2^p - 1)    % Рунгеова оцена грешке
```

> **Зашто `3:4:n` и `5:4:n`?** На грубој мрежи са кораком $2h$ и $n=8$, тачке које добијају коефицијент 4 у Симпсону су $x_2$ и $x_6$ — у MATLAB табели то су индекси 3 и 7, тј. `3:4:8`. Тачка са коефицијентом 2 је $x_4$ — индекс 5, тј. `5:4:8`.

## 7. АУТОМАТСКО ОДРЕЂИВАЊЕ БРОЈА ПОДИНТЕРВАЛА

**Идеја (Задатак 3):** Удвостручавамо $n$ све dok Рунгеова оцена не падне испод задате тачности $\varepsilon$.

```matlab
f = @(x) (1 - x.^2) ./ (x + x.^3);
a = 1;  b = 2;
tol = 0.5e-4;    % тражена тачност
p = 4;           % Симпсон
n = 2;           % почињемо са n=2 (најмање паран)

while true
    % израчунај Is за тренутно n
    X = linspace(a, b, n+1);   h = (b-a)/n;
    F = zeros(n+1,4);
    F(:,1)=X; F(1,2)=f(X(1)); F(n+1,2)=f(X(n+1));
    F(2:2:n,3)=f(X(2:2:n)); F(3:2:n,4)=f(X(3:2:n));
    Ihs = h/3*(F(1,2)+F(n+1,2)+4*sum(F(:,3))+2*sum(F(:,4)));

    % израчунај Is за корак 2h (груба мрежа)
    I2hs = (2*h)/3*(sum(F(:,2))+4*sum(F(3:4:n,4))+2*sum(F(5:4:n,4)));

    E = abs(Ihs - I2hs) / (2^p - 1);   % Рунгеова оцена

    if E < tol
        break        % тачност достигнута — излази из петље
    end
    n = 2*n;         % удвостручи број подинтервала
end

fprintf("n = %d,  I ≈ %.6f,  E ≈ %.2e\n", n, Ihs, E);
```

## 8. Брза референца — поређење метода

| Метода | Апроксимација | Степен тачности | Ред грешке | Оцена грешке |
|---|---|---|---|---|
| Правоугаоник | константа | 1 | $O(h)$ | $\frac{h}{2}(b-a)M_1$ |
| Трапез | права линија | 1 | $O(h^2)$ | $\frac{(b-a)}{12}h^2 M_2$ |
| Симпсон | парабола | 3 | $O(h^4)$ | $\frac{(b-a)}{180}h^4 M_4$ |

> Симпсон удвостручавањем $n$ смањује грешку 16×, трапез само 4×, правоугаоник само 2×.
