# 📐 Полиномска Интерполација — Cheatsheet

> Сви кодови за Лагранжову и Њутнову интерполацију у Матлабу.

---

## 1. ЛАГРАНЖ — Вредност у тачки (даје број)

```matlab
X = [0 1 3 4];       % чворови
f = [1 3 2 1];       % вредности функције
x0 = 3.5;            % тачка у којој тражимо вредност
n = length(X);

P = 0;
for i = 1:n
    L = 1;
    for j = 1:n
        if j~=i
            L = L*(x0 - X(j))/(X(i)-X(j));
        end
    end
    P = P + L*f(i); %f(i) ako je f dat kao polinom
end
fprintf("Priblizna vrednost f(%.4f) = %.6f", x0, P);
```

**Резултат:** Један број — вредност f(x0).

---

## 2. ЛАГРАНЖ — Формирање полинома (даје вектор коефицијената) 

```matlab
X = [0 1 3 4];
f = [1 3 2 1]; %npr da je f bilo x+1 onda koristimo F=f(X) da dobijem pol i tako ide u petlju
x0 = 3.5;
n = length(X);

Pol = 0;
for i = 1:n
    L = 1;
    for j = 1:n
        if j~=i
            L = conv(L, [1 -X(j)]) / (X(i)-X(j));
            % conv множи полиноме, [1 -X(j)] = (x - X(j))
        end
    end
    Pol = Pol + L*F(i); %ako je f funkc i F polinom
end
Pol   % вектор коефицијената нпр. [a b c d] = ax^3 + bx^2 + cx + d

fprintf("f(%.4f) = %.6f", x0, polyval(Pol, x0));
```

**Разлика vs Код 1:** `L` је вектор (полином) уместо броја, `conv` уместо `*`.

---

## 3. ЛАГРАНЖ — Процена грешке

```matlab
% (покренути после формирања Pol-а)
f_sym = sym(@(x) ...);     % овде упиши своју функцију
dfn = diff(f_sym, n);      % n-ти извод
dfn = matlabFunction(dfn);

M = max(abs(dfn(X(1))), abs(dfn(X(end))));  % max izvoda na krajevima

R_help = 1;
for i = 1:n
    R_help = R_help * abs(x0 - X(i));
end

R = M / factorial(n) * R_help;
fprintf("Ocena greske: |Rn(x)| <= %.7f", R);
```

**Формула:** $|R_n(x)| \leq \frac{M_{n+1}}{(n+1)!} \cdot |(x-x_0)(x-x_1)\cdots(x-x_n)|$

## 4. ЊУТН — Табела подељених разлика

```matlab
X = [1.0 1.1 1.3 1.5 1.6];
f = @(x) x.^(1/3);
F = f(X);
n = length(X);

A = zeros(n, n+1);
A(:,1) = X;      % 1. колона: чворови
A(:,2) = F;      % 2. колона: f(xi) — нулти ред

for j = 1:n-1          % j = ред подељених разлика
    for i = 1:n-j      % i = редни број у том реду
        A(i,j+2) = (A(i+1,j+1) - A(i,j+1)) / (X(i+j) - X(i));
    end
end
disp(A);
% Користимо САМО prvi red (i=1): A(1,2), A(1,3), A(1,4), ...
```

**Формула:** $f[x_i, x_{i+1}] = \frac{f(x_{i+1}) - f(x_i)}{x_{i+1} - x_i}$, и тако рекурзивно.

---

## 5. ЊУТН — Формирање полинома

```matlab
% (покренути после табеле подељених разлика)
x0 = 1.15;

P = A(1,2);   % почињемо са f[x0]
Q = 1;        % Q прати производ (x-x0)(x-x1)...

for j = 1:n-1
    Q = conv(Q, [1 -X(j)]);     % Q = Q * (x - xj)
    P = [0 P] + Q * A(1,j+2);  % додајемо novi clan
    % [0 P] повећава степен P за 1 да би сабирање радило
end
P
fprintf("f(%.4f) = %.5f", x0, polyval(P, x0));
```

**Формула:** $P_n(x) = f[x_0] + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + \cdots$

---

## 6. ЊУТН — Процена грешке

```matlab
f_sym = sym(f);
for i = 1:n
    df = diff(f_sym, i);
end
df = matlabFunction(df);

M = max(abs(df(X(1))), abs(df(X(end))));
Q_err = conv(Q, [1 -X(n)]);   % Q * (x - xn)
R = abs(M / factorial(n) * polyval(Q_err, x0));
fprintf("Ocena greske: |R%d(x)| <= %.7f", n, R);

if tol < R
    fprintf("Nije ispunjena trazena tacnost — dodaj novi cvor!");
end
```

## 7. ЊУТН — Додавање новог чвора ⭐

> Ово је главна предност Њутна над Лагранжом!

```matlab
% ============================================================
% ПРИМЕР: X = [0.8, 1.2, 1.5], тачност није довољна
% Додајемо нови чвор 1.3
% ============================================================

% СТАРИ код завршава са:
%   P  = полином са 3 чвора (степен 2)
%   Q  = (x-0.8)(x-1.2)(x-1.5)  ← Q из петље + задnja linija greske

% НОВИ чвор:
X = [0.8 1.2 1.5 1.3];   % dodajemo 1.3 na kraj
f = @(x) exp(-x);
F = f(X);
n = length(X);            % sada n=4

% Поново правимо tabelu (mora za novi cvor):
A = zeros(n, n+1);
A(:,1) = X;
A(:,2) = F;
for j = 1:n-1
    for i = 1:n-j
        A(i,j+2) = (A(i+1,j+1) - A(i,j+1)) / (X(i+j) - X(i));
    end
end

% ⭐ КЉУЧНА РАЗЛИКА — само dodajemo jedan novi clan:
P = [0 P] + Q * A(1, n+1);
%    ↑          ↑
%  stari P   Q iz prethodnog koraka (vec sadrzi sve (x-xi))
%             A(1,n+1) = nova podeljena razlika f[x0,x1,x2,x3]

fprintf("f(%.6f) = %.6f", x0, polyval(P, x0));
```

### Шта се мења кад додамо чвор:

| | Без новог чвора | Са новим чвором |
|---|---|---|
| `n` | 3 | 4 |
| `X` | `[0.8 1.2 1.5]` | `[0.8 1.2 1.5 1.3]` |
| Табела `A` | 3×4 | 4×5 |
| Полином `P` | степен 2 | степен 3 |
| Код за P | цела петља | само `[0 P] + Q*A(1,n+1)` |
| `Q` | `(x-0.8)(x-1.2)(x-1.5)` | исти (из оцене грешке) |

> **Зашто не правимо P од нуле?** Зато што Њутнов полином има особину да нови члан само `ДОДАЈЕМО` на крај — претходни чланови остају непромењени.

## 8. Брза референца — Кад користити шта?

| Ситуација | Метода |
|---|---|
| Само треба вредност у једној тачки | Лагранж Код 1 |
| Треба цео полином (за цртање, вишеструко вредновање) | Лагранж Код 2 или Њутн |
| Можда ће требати додати чворове | **Њутн** (Лагранж не може лако) |
| Дата табела са неједнаким размацима | Оба раде |
